ผมมีโครงการจะพัฒนาโปรแกรม Project Name: Smart Cash Flow

เป้าหมาย: เก็บข้อมูลจากธนาคารหลายแห่งที่ส่งไฟล์มาเป็น pdf นำมาแปลงเป็นข้อมูลลง excel เพื่อสรุปเงินรับเข้าจ่ายออกของแต่ละธนาคาร
การทำงาน: ใช้ ocr และ extract data ออกมาเป็น excel ตามรูปแบบที่กำหนด
วิธีการพัฒนาต้องใช้ Antigravity ช่วยพัฒนา 
**ต้องการให้ช่วยเขียน spec. เพื่อ AI เข้าใจนำไปพัฒนาต่อได้ง่าย(project-context.md)
- black-end: ช่วยจัดการไฟล์ pdf สำหรับเตรียมทำ ocr -> ทำ ocr (ด้วยเครื่องมือ 2 แบบให้เลือกคือ Typhoon และ tesseract) -> extract
ไปเรียงใน excel ดังนี้
    1.เลขที่บัญชี หา "A/CNO:" แล้วดึงเลขที่เช่น 9943-000613-001
	2.วันที่เอกสาร หา "Date" แล้วดึงวันที่ที่อยู่บนสุดของเอกสาร เช่น 01-Dec-2025 หรือรูปแบบอื่นๆที่นิยมทั่วไป และแปลงเป็น dd/MM/yyyy (ปี ค.ศ.)
	3.เลขที่อ้างอิง หา "Our Ref" หรือคำภาษาอังกฤษอื่นๆที่นิยมใช้งาน แล้วดึงเลขที่มา เช่น IC 25/0660
	4.มูลค่ารวม หา "Total Debited" หรือคำอื่นๆที่ใกล้เคียง แล้วดึงมูลค่ามาเช่น 165,758.51 (มูลค่ารวมอาจไม่มี label ขึ้นต้นดึงจากตัวเลขที่อยู่ท้ายสุดของหน้า
	5.ธนาคาร ใช้วิธี lookup โดยใช้ เลขที่บัญชี ไปดึงข้อมูลจากไฟล์ excel name: AC_Master ถ้าเลขที่บัญชีตรงกันดึง ชื่อธนาคมา
	6.บัญชีบริษัท ใช้วิธี lookup โดยใช้ เลขที่บัญชี ไปดึงข้อมูลจากไฟล์ excel name: AC_Master ถ้าเลขที่บัญชีตรงกันดึง ชื่อบริษัท
	**เมื่อได้ข้อมูลครบส่งไปวางใน excel เรียงลำดับ column 1-6 (เพื่อเป็นฐานข้อมูลสำหรับออกรายงานต่อไป)
	
- front-end: หน้าจอสำหรับเลือก folder PDF ที่จะทำ ocr มีการทำงานดังนี้
	1.เลือก folder ที่ต้องการทำ ocr
	2.มีปุ่ม run โปรแกรม back-end โดยอ่านทุก files และทุกหน้าใน folder เลือก
	3.แสดงผล column ที่ extract ได้(สามารถ edit)
	4.มีหน้าจอสำหรับกดเรียกรายงานในรูปแบบต่างๆ (จะเพิ่มเติมในอนาคต)
	**หน้าจอมีรูปทันสมัย ดูสบายตา ใช้งานง่าย
